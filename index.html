Cod Ta'rif
git config --list git fayllari xaqida ma'lumot beradi
git status xolatni bilish
git add gitga fayl qo'shish
git add . bo'lsa barcha fayllarni gitga qo'shish
git commit -m "commitni nomlash" yuborilayotgan o'zgarishni nomlash
git reset index.html oxirgi qilgan o'zgartirishni ortga qaytarish
git log Bu gitga o'zgartirish kiritganlik xaqida ma'lumot beradi (kim o'zgartirish kiritdi, soati, commitni nomi...)
git checkout [git id si] commit qilingan qaysidur xolatga qaytish uchun
git checkout main qaysi branchga o'tishni tanlash uchun
git branch biz ayni vaqtda qaysi branchda turganimizni ko'rsatadi
git branch [heder] Bu yangi branch ochish
git push -u origin [branch nomi] Bu proektni push qilish uchun
Gitlens extantion git uchun qatorlarda git xaqida ma'lumot beradi
Git Graph extantion git uchun (Bu branch xolatini chizib beradi)
git merge [header] Bu branchni birlashtirish uchun (asosiy branchga o'tib qo'shimcha branchni chaqiriladi)
